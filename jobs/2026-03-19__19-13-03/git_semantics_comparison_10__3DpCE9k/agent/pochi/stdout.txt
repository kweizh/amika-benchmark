Use `jj bookmark set` to move/create bookmarks, then `jj edit` to make the target commit the working copy.

From `/home/user/repo`:

```sh
cd /home/user/repo
jj bookmark set feature -r 'description("third")'
jj bookmark set bugfix -r 'description("second")'
jj edit 'description("second")'
```

What this does:
- Moves `feature` to the commit described as `third`
- Creates `bugfix` at the commit described as `second`
- Makes `second` the current working-copy commit

If you want to verify afterward, run:

```sh
jj log
jj bookmark list
```
