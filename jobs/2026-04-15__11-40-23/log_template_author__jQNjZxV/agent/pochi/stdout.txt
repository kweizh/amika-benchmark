
**Breakdown of the command:**

| Part | Purpose |
|---|---|
| `jj log` | Lists commits |
| `-r @` | Restricts output to the current working copy commit |
| `--no-graph` | Suppresses the revision graph (only raw template output) |
| `-T '...'` | Specifies the template string |

**Template details:**

- `author.name()` — accesses the `name()` method of the `Signature` type returned by `author` on the implicit `Commit` object
- `author.email()` — accesses the `email()` method of the same `Signature`
- `++` — string concatenation operator in jj's template language
- `"\n"` — trailing newline for exact formatting
