# Create a Remote Sandbox with Amika

## Background
You have the Amika CLI installed in your environment and are already authenticated. You need to create a new remote sandbox from the current git repository.

## Requirements
- Create a new remote sandbox using the Amika CLI.
- The sandbox name must be `harbor-sandbox-test`.
- The sandbox must be created from the current git repository and be a remote sandbox.

## Constraints
- Project path: /home/user/workspace
- Use Amika CLI for sandbox creation.
