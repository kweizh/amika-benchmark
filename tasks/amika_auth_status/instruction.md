# Verify Amika Auth Status

## Background
Amika provides a CLI for managing sandboxes. Users must be authenticated to interact with Amika Cloud. A valid API key is already configured in the environment.

## Requirements
- Run the `amika auth status` command to check the current authentication status.
- Save the exact output of the command to `/home/user/output.log`.

## Constraints
- Log file: `/home/user/output.log`
- Project path: `/home/user`
