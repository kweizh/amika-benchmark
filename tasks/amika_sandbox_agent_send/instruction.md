# Amika Sandbox Agent Send

## Background
You have the Amika CLI installed and are authenticated. Amika allows you to send instructions directly to an agent CLI inside a persistent sandbox.

## Requirements
- Create a remote sandbox named `agent-box` from the current directory.
- Send a prompt to the agent inside `agent-box` to create a file named `hello.txt` with the content `Hello, Amika!` in the current directory.

## Constraints
- Project path: /home/user/project
- Use Amika CLI for all operations.
